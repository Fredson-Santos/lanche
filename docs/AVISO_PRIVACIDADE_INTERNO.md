# Aviso de Privacidade Interno - Sistema LANCHE

Este documento descreve como o sistema **LANCHE** trata os dados pessoais dos seus funcionários e colaboradores, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

## 1. Finalidade do Tratamento
O tratamento de dados pessoais no sistema LANCHE tem como finalidades exclusivas:
- **Gestão Operacional**: Identificação do operador responsável por vendas, movimentações de estoque e ordens de reposição.
- **Segurança e Auditoria**: Registro de logs de atividades para prevenir fraudes, erros operacionais e garantir a integridade dos dados financeiros.
- **Cumprimento de Contrato**: Execução das atividades laborais pactuadas entre o colaborador e a empresa.

## 2. Dados Coletados
O sistema coleta e armazena os seguintes dados:
- **Identificação**: Nome de usuário (username) e e-mail corporativo.
- **Atividade**: Registro de data, hora e tipo de transação realizada (Vendas, Alertas, Reposição).
- **Acesso**: Endereço IP e informações do dispositivo de acesso para fins de segurança.

## 3. Base Legal
O tratamento destes dados fundamenta-se no **Art. 7º, inciso V da LGPD**: "quando necessário para a execução de contrato ou de procedimentos preliminares relacionados a contrato do qual seja parte o titular, a pedido do titular dos dados".

## 4. Direitos do Titular
Como usuário do sistema, você tem direito a:
- **Confirmação e Acesso**: Saber quais dados seus estão sendo tratados. Você pode consultar seu perfil e histórico de atividades diretamente no sistema através do seu painel de usuário.
- **Correção**: Solicitar a correção de dados incompletos, inexatos ou desatualizados.
- **Informação**: Saber com quem seus dados são compartilhados (apenas administradores autorizados da empresa).

## 5. Proteção de Dados
A empresa adota medidas técnicas e administrativas para proteger seus dados, incluindo:
- Criptografia de dados sensíveis em repouso.
- Monitoramento contínuo de logs de acesso.
- Controle de acesso baseado em funções (RBAC).

# Atividade Prática Integradora

**Data de conclusão:** 22 de abril de 2026 às 23:59

## Instruções
Com base nos cenários disponibilizados nos anexos, cada aluno ou grupo deverá escolher **1 cenário** para desenvolver uma solução completa de sistema, contemplando front-end, back-end e documentação de Engenharia de Software.

Os cenários disponíveis são: SALA, REMEDIO, **LANCHE** e LIVRO.

---

## Requisitos do Projeto

### 1. Desenvolvimento do Sistema
- [ ] **Front-end funcional**
- [ ] **Back-end funcional**
- [ ] **Integração entre as camadas**
- [ ] **Banco de dados** compatível com a proposta do cenário escolhido

### 2. Documentação do Projeto
- [ ] Levantamento de Requisitos Funcionais (RF)
- [ ] Levantamento de Requisitos Não Funcionais (RNF)
- [ ] Regras de Negócio (RN)
- [ ] MER – Modelo Entidade-Relacionamento
- [ ] DER – Diagrama Entidade-Relacionamento
- [ ] Modelagem do banco de dados
- [ ] Descrição da arquitetura da solução
- [ ] Demais artefatos pertinentes à disciplina de Engenharia de Software
